Pardus 2008.2 Verionshinweise

1. Willkommen zu Pardus 2008.2!

Willkommen zu Pardus Version 2008.1, einer weiteren grossartigen Pardus Version von Pardus Entwicklern, Übersetzern, Designern und Dokumentationsverfassern. Dies ist die zweite Unterversion von der Pardus 2008 Reihe.
Für weitere Informationen bitten wir Sie http://www.pardus.org.tr/eng zu besuchen

1.1 Feedback

Sie können http://bugs.pardus.org.tr, alias Fehler-Verfolgungs-System (Bug Tracking) benutzen, um einen Fehler, eine Fehlerlösung oder Verbesserungsvorschläge im Zusammenhang mit Pardus zu melden. Wir bedanken uns an dieser Stelle bei den Benutzern und Entwicklern, für die ihre vielfältigen Meldungen, Lösungen und Vorschlägen die zu dieser aktuellen Ausgabe beigetragen haben.

2. Haupteigenschaften

Since the latest minor release 2008.1, more than 120 bug fixes and 60 security fixes were committed to 648 packages included in the installation CD. In the same period, 23 new packages were added to the CD to support new featues and hardware, and a total of 181 (out of 648) packages were updated.

2.1. Updates in Pardus Technologien

Mit der Version Pardus 2008.2 sind Kerntechnologien wie PiSi und COMAR aktualisiert um neue Features bereitzustellen. YALI ist ebenso aktualisiert um eine reibungslosere und zuverlässigere Installationserfahrung anzubieten. Die Verbesserungen im Mudur-Initialisierungssystem ermöglicht einen schnelleren und robusteren Bootvorgang mit einem besseren RemoteFileSystem-Support.

  • Netzwerk-Manager - Pardus' eigenes Tool fir die Verwaltung der Verbindungen ist aktualisiertund enthält nun eine Reihe von neuen Feautres:
    - DHCP related timeout enhancements,
    - Support for Non-ASCII ESSID names,
    - Flexible DNS settings,
    - Support for script execution upon connection and disconnection.

  • Anzeige-Manager - Pardus 2008.2 includes the latest drivers for new generation Nvidia, ATI and Intel based display adapters. Pardus' own Display Manager and COMAR based auto-configuration subsystem zorg are updated to make use of these new drivers and features.

  • Paket-Manager - The Package Manager in Pardus 2008.2 includes a bunch of updates as well :
    - Improved package search function,
    - Support for handling download errors and retries mostly caused by proxies and such,
    - GUI and translation updates,
    - Delta packages and automatic updates of reverse dependencies - these features are planned to be supported after the release of 2008.2.

2.2. KDE 3 und KDE 4 Desktop-Umgebungen

  • KDE 3 - KDE 3, the default desktop environment of Pardus 2008.2, is updated and enhanced with Pardus specific features as well. These updates are based on the latest stable release of KDE 3, 3.5.10. Pardus 2008.2 Installation CDs and KDE 3 based Live CD include KDE 3 platform by default.

  • KDE 4 - New member of the KDE family, KDE 4, has been "the" choice of many Pardus users for a while now. Pardus 2008.2 includes the latest stable release of KDE 4, 4.2 that has been released a few days ago. Users who prefer this new desktop environment can install and use KDE 4.2 with the release of 2008.2. The KDE 4 based Live CD includes this new desktop environment and the KDE 4 ports of popular KDE applications as well.

2.3. Zwei Varianten von Installations-CDs

Pardus is offered as two variants to support ever-increasing number of languages : One installation CD with support for Turkish and English only (and a selection of additional development tools and games), and another internationalized installation CD with support for all of the 11 languages. As always, users can install additional language support packages from Pardus repositories, regardless of which installation CD they've used to install their systems.

2.4. Aktualisierte Komponente

Pardus 2008.2 enthält eine Reihe von Paket-Updates zuden Pardus Technologien und KDE Komponenten. Einige von den Highlights sind:

  • Pardus Linux Kernel in 2008.2 is based on the latest stable release of linux 2.6.25, 2.6.25.20. Pardus kernel also includes a lot of hardware driver and subsystem updates, stability and security fixes. Some of these new features are :
    - Fixes and enchanced support for network devices based on RTL818x, Marvell Yukon 88E8040T and Ralink rt2x00 chipsets,
    - IR controller for bt8xx based TV cards,
    - Fixes and enchanced support for Syntek DC-1125 based webcams,
    - Fixes for Asus eeePC, Apple Macbook Pro and Gigabyte M9xx,
    - Support for Broadcom BCM4311, BCM4312, BCM4321 and BCM4322 based wireless devices,
    - Support for Intel WiFi Link 5000 based wireless devices,
    - Support for RTL8187SE based wireless devices,
    - Support for RICOH Bay1Controller RICOH SD card readers,
    - ALSA 1.0.18a_20081203 contains support and fixes for a bunch of soundcards,
    - Support for splash images in different resolutions,
    - Support for init images with virtio and xen modules.

  • Also with Mozilla Firefox 3.0.5, OpenOffice.org 2.4.1.10, Mplayer 20090101 and the updated popular applications like KDEPIM, Vim, Gimp, Frozen Bubble; Pardus 2008.2 is more stable, secure and user-friendly as ever.


For more information about updates and new features on Pardus 2008.2, you can take a look at the detailed changelog on our web site.

3. Installationsanforderungen

Für die Installation und Verwendung von Pardus 2008.2 wird ein Rechner mit mindestens 512 MB RAM (Arbeitsspeicher) und ein Prozessor mit 1400 MHz empfohlen. Minimale und empfohlene Hardwarekomponente für Pardus 2008.2:

  • 384 MB Arbeitsspeichers (512 MB empfohlen)

  • 1000 MHz Intel oder AMD Prozesseor (1400 MHz empfohlen)

  • Min. 4 GB freien Festplattenspeicher (10 GB empfohlen) * Wir empfehlen für die KDE4-basierte LiveCD ein System mit min. 1 GB Arbeitsspeicher.
