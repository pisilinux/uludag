Notas de Lanzamiento Pardus 2008.2

1. Bienvenido a Pardus 2008.2!

Bienvenido a Pardus 2008.2, otra versión estupenda preparada por los desarrolladores de Pardus, traductores, diseñadores y editores de documentación. Esta es la segunda actualización de la serie Pardus 2008.
Le invitamos cordialmente a visitar http://www.pardus.org.tr/eng por más información.

1.1 Notificación - Feedback

Puede entrar en http://bugs.pardus.org.tr, , alias Pardus "bug tracking system" para enviar un reporte de falla, una corrección o cualquier sugerencia relacionada con Pardus. Nos gustaría agradecer en este lugar a nuestros usuarios y desarrolladores por el envío de sus reportes de fallas, correcciones y sugerencias para nuevos paquetes y nuevas caracteristicas de esta versión.

2. Características principales

Desde la ultima versión de actualización 2008.1, se realizaron más de 120 correcciones y se aplicaron 60 correcciones de seguridad para los 648 paquetes, que están incluidos en el CD de instalación. En el mismo período fueron agregados 23 paquetes nuevos al CD, para soportar nuevas características y hardware. Además, 181 paquetes (de los 648) fueron actualizados.

2.1. Actualización en Tecnologías de Pardus

Las tecnologías específicas de Pardus, como PiSi y COMAR, fueron actualizados en la versión Pardus 2008.2 y proveen facilidades nuevas. También YALI fue actualizado, para una experiencia de instalación más agradable y confiable. Las mejoras en el sistema de inicialización Mudur resultan en un proceso de inicialización más rápido y más robusto, además con mejor soporte para sistemas de archivos remotos.
  • Administrador de Red (Network Manager) - El administrador de conexiones de red, propio de Pardus, fue mejorado con varias facilidades adicionales:
    - Mejor manejo de timeout con DHCP,
    - Soporte para nombres ESSID No-ASCII,
    - Configuración DNS flexible,
    - Soporte para la ejecución de scripts al conectar o desconectar.
  • Administrador de Pantallas (Display Manager) - Pardus 2008.2 incluye los ultimos controladores para las nuevas generaciones de adaptadores de video basados en Nvidia, ATI e Intel. Display manager y el subsistema zorg para auto-configuración de Pardus, basado en COMAR, fueron actualizados para aprovechar las nuevas versiones de controladores y las facilidades nuevas.
  • Administrador de Paquetes (Package Manager) - También el Administrador de Paquetes de Pardus 2008.2 incluye una serie de mejoras:
    - Mejorada función de búsqueda de paquetes;
    - Soporte para manejo de errores de descarga y reintentos, mayormente por causa de algún proxy;
    - Actualizaciones de la interfaz GUI y traducciones;
    - Paquetes Delta y actualizaciones automáticas de dependencias reversas, previstos en una actualización posterior al lanzamiento de Pardus 2008.2.

2.2. Entornos de Escritorio KDE 3 y KDE 4

  • KDE 3 - KDE3, el escritorio por defecto de Pardus 2008.2, fue actualizado e incluye adaptaciones especiales para Pardus. Las actualizaciones están basados en la última versión estable de KDE 3, 3.5.10. Los CDs de instalación de KDE 3 y los Live CD de Pardus 2008.2 incluyen esta versión de KDE 3 como entorno por defecto.
  • KDE 4 - El nuevo miembro de la familia KDE, el KDE 4, ya está en uso por muchos usuarios de Pardus por un tiempo. Pardus 2008.2 incluye la ultima versión estable de KDE 4, 4.2 - lanzado hace pocos días. Usuarios que desean usar este entorno de escritorio nuevo, pueden instalar y usar KDE 4.2 con la versión Pardus 2008.2. El Live CD con KDE 4 incluye el nuevo entorno de escritorio y las versiones de varias aplicaciones KDE portadas a la versión KDE 4.

2.3. Dos variantes de CDs de Instalación

Se ofrece Pardus en dos (2) variantes, por el soporte de idiomas, que está aumentando:
  • Un CD de instalación con soporte exclusivamente para Turco e Inglés (incluye además una selección adicional de herramientas de desarrollo, y de juegos), y
  • otro CD de instalación internacional, con soporte para los demás 11 idiomas.
Sin embargo, como siempre, se puede agregar y utilizar soporte para idiomas adicionales con los paquetes de los repositorios de Pardus, independiente del CD de instalación utilizado para la instalación del sistema.

2.4. Componentes Actualizados

Pardus 2008.2 incluye muchas actualizaciones, aparte de actualizaciones tecnologicas específicas de Pardus o de KDE.
Algunas destacados son:
  • El Kernel de Pardus 2008.2 está basado en la ultima versión estable de linux 2.6.25: 2.6.25.20. El kernel de Pardus también incluye muchos controladores de hardware, y actualizaciones de subsistemas, estabilidad y correcciones de seguridad. Algunas de las caracteristicas nuevas son:
    - Correcciones y soporte mejorado para dispositivos de red basados en chipsets de RTL818x, Marvell Yukon 88E8040T y Ralink rt2x00,
    - Controlador IR para tarjetas de TV basados en bt8xx,
    - Correcciones y soporte mejorado para webcams basados en Syntek DC-1125,
    - Adaptaciones para Asus eeePC, Apple Macbook Pro y Gigabyte M9xx,
    - Soporte para dispositivos inalambricos basados en Broadcom BCM4311, BCM4312, BCM4321 y BCM4322,
    - Soporte para dispositivos inalambricos basados en Intel WiFi Link 5000,
    - Soporte para dispositivos inalambricos basados en RTL8187SE,
    - Soporte para lectores de tarjetas SD con RICOH Bay1Controller,
    - ALSA 1.0.18a_20081203 tiene correcciones y soporta más tarjetas de sonido,
    - Soporte para imagenes splash de varias resoluciones,
    - Soporte para imagenes init con módulos virtio y xen.
  • Además con Mozilla Firefox 3.0.5, OpenOffice.org 2.4.1.10, Mplayer 20090101 y actualizaciones de aplicaciones populares como KDEPIM, vim, Gimp, Frozen Bubble; Pardus 2008.2 es más estable, seguro y amigable que nunca.

Por más información sobre las actualizaciones y nuevas características en Pardus 2008.2 puede visitar histórico detallado de cambios en nuestro sitio web.

3. Requerimientos de instalación

Puede ejecutar Pardus 2008.2 en un equipo con un hardware regular. Sugerimos al menos 512 MB de memoria RAM y un procesador de 1400 MHz para un buen rendimiento. Requisitos de hardware mínimos y recomendados para Pardus 2008.2 son:
  • 384 MB memoria RAM (512 MB recomendado)
  • Procesados Intel o AMD de 1000 MHz  (1400 MHz recomendado)
  • Al menos 4 GB de espacio en disco duro (10 GB recomendado)
    • * Recomendamos un mínimo de 1 GB RAM para el Live CD con KDE 4 (CD booteable).
