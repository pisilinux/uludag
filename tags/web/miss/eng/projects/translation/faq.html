Translation FAQ

Q: I'm not a developer. What is SVN?

   SVN is used to send and receive files on Pardus server. In order to translate po files, you need an SVN account on the server.

Q: How can I apply for a translation?

    Read "Applying for translation".

Q: Can you open a mailing list for my translation team?

   Yes. Just request for a mailing list from admins@pardus.org.tr after you have applied for an SVN account. Your language mailing list is in the form of pardus-translation-XX where XX is the ISO code of your language (i.e nl for Dutch, it for Italian etc).
