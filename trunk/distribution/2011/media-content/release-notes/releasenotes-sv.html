Versionsfakta

1. Välkommen till Pardus 2011

Välkommen till Pardus 2011, den senaste utgåvan av Pardus, en kraftfull Linuxdistribution fokuserad på KDE.

1.1. Vad är Pardus?

Pardus är öppenkällkod, fritt distribuerbar, GPL (General Public License)-licensierad användarvänlig GNU/Linuxdistribution med många förbättringar inom användbarhet, hårdvarukompatibilitet, stabilitet och säkerhet.
Pardusprojektet sponsras av Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), ett dotterbolag till Scientific and Technological Research Council of Turkey (TUBITAK).
Målet med Pardusdistributionen är att erbjuda människor en stabil, pålitlig, lättanvänd och innehållsrik datorupplevelsegenom att erbjuda moderna och populära program som har öppen källkod. Utvecklarna lägger också extra kraft på att få Pardus körbar på ett brett spektrum av hårdvara.
Projektet är döpt efter det latinska och vetenskapliga namnet på den anatoliska leoparden, Panthera pardus tulliana.

1.2. Hur kan jag testa Pardus?

Du kan testa Pardus antingen genom att installera det på din dator eller köra det live utan att påverka din hårddisk. Du kan starta Pardus från en optisk enhet eller från USB-enheter såsom hårddiskar och minnepinnar.
Både installationen och liveversionens avbilder överstiger vad som får plats på en vanlig CD, om du vill använda en optisk enhet för att starta Pardus måste du för närvarande använda en DVD.
Tänk på att uppstartsordningen i datorns BIOS-program måste ställas in till att först starta från optisk enhet eller USB-enhet för att start av live eller installation skall ske. På nyare system kan uppstartsordningen också ställas in genom att trycka på en tangent, vanligtvis F8, under uppstart.

1.3. Mer information

Besök den officiella webbplatsen http://www.pardus.org.tr/eng för mer information.

1.4. Återkoppling

Besök http://bugs.pardus.org.tr för att rapportera fel, skicka in en felrättning och föreslå förbättringar av Pardus. Vi vill tacka alla användare och medlemmar av gemenskapen för deras värdefulla bidrag som hjälpt till att förbättra Pardus, Linux och fri programvara världen över.

2. Översikt

Pardus 2011 innehåller stora förbättringar jämfört med föregående Pardusutgåva 2009.2. Här följer en kort sammanfattning över dessa förbättringar.

2.1. Hårdvara

Pardus 2011 har stöd för 64-bitars processorarkitektur som stöds av många nyare Intel- och AMD-processorer.

2.2. Program

  • Kärnan - Den senaste Linuxkärnan 2.6.37 tillhandahåller det senaste hårdvarustödet tillsammans med tusentals felrättningar.
  • Plymouth - Bootsplash-tekniken som användes i Pardus 2009.2 är helt ersatt med Plymouth-motorn.
  • YALI - YALI, installationsprogrammet för Pardus, har fått stöd för LVM/RAID och UUID.
  • KDE SC - Pardus 2011 comes with the latest KDE Software Compilation, KDE SC 4.5.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
  • Kaptan - Kaptan, anpassningsverktyget för Pardusskrivbordet, nu med möjligheten att ta din bild och använda som avatar i KDE.
  • NetworkManager - GNOME NetworkManager 0.8.2 är nu standardhanteraren för nätverksbakänden i Pardus 2011. Användaren kan nu ställa in sina nätverk för HSPA/CDMA/VPN tillsammans med det redan stödda Ethernet och 802.11 WLAN-nätverk.
  • GTK Oxygen style - Alla GTK-program ritas i stilen Oxygen tack vare oxygen-gtk-projektet. Detta leder till en enorm förbättring till konsekvens i användargränssnittet.
  • LibreOffice - LibreOffice, en kontorssvit baserad på öppen källkod sponsrad av Document Foundation, är nu standard i Pardus 2011.

3. Vad är nytt?

3.1. Uppdateringar av Pardustekniker

3.1.1. YALI

  • YALI, installationsprogrammet för Pardus, har fått stöd för LVM/RAID och UUID.
  • Användargränssnittet är helt omgjort för att öka användbarheten.
  • Användaren kan nu välja mellan mellan drivrutin med öppen eller sluten källkod för NVIDIAs grafikkort.

3.1.2. Första starten

  • Vissa inställningar som efterfrågades under installationen av Pardus 2009.2 är nu flyttade till ett annat program som körs en gång under den första uppstarten efter installationen.

3.1.3. Kaptan

  • Kaptan, anpassningsverktyget för Pardusskrivbordet, nu med möjligheten att ta din bild och använda som avatar i KDE.
  • Användarna kan nu välja sitt önskade ikontema som används i KDE.
  • Användargränssnittet är helt omgjort för att öka användbarheten.

3.1.4. Pisi

  • Pisi använder nu XZ som standard packningsalgoritm.
  • Binary package names now reflects the architecture and the distribution release for which the package is built.
  • Various speed enhancements has been done in the core components for a better package management experience.

3.1.5. Pakethanteraren

  • Users are now able to vote for a package and see the screenshots of graphical applications shipped within packages.
  • The graphical user interface is redesigned to improve the usability. Various speed enhancements has been done for a better graphical package management experience.
  • A lot of KDE integration work has been done including the usage of KDE system tray.

3.2. Uppdateringar av kärnkomponenter

  • Kernel - The Linux kernel shipped with Pardus 2011 is 2.6.37 which is the latest stable kernel released on 2011/01/05.
  • Plymouth - The bootsplash technology used in Pardus 2009.2 is completely dropped and replaced by the new Plymouth engine.
  • Python - Python is updated to 2.7.1.
  • Perl - Perl is updated to 5.12.2.
  • X.org - xorg-server is updated to 1.9.4 RC1, improvements to the automatic driver configuration mechanism has been done by our developers.
  • TeX Live - TeX Live documentation stack is updated to 2009.

3.3. Skrivbordsupplevelse

3.3.1. Changes in KDE Desktop Environment

Pardus 2011 comes with the latest KDE Software Compilation, KDE SC 4.5.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE SC 4.5.5 brings more performance and stability to the desktop experience. Support for WebKit, improved user experience for netbooks, a new high-speed cache allowing fast access to resources stored on disk are among the new features of this version.

3.3.2. Visual Enhancements

All GTK applications are now rendered with Oxygen style thanks to the oxygen-gtk project. This brings a huge improvement to the user interface consistency.
Milky icon theme especially designed for Pardus is shipped within the installation image. Users can set this icon theme through Kaptan or System Settings.

3.3.3. Nätverkshantering

Besides switching to GNOME NetworkManager as the default networking backend, Pardus 2011 comes with a heavily improved and modified KDE Network Management tool and plasmoid to give their users a much more consistent networking experience.
With this new networking backend, users are now able to easily set up their Ethernet (With support for 802.1x authentication), 802.11 Wireless (With support for WEP/WPA/WPA2/WPA-EAP and 802.1x authentication), EDGE/3G/CDMA and VPN networks.

3.3.4. Förvalda program

3.3.4.1. Mozilla Firefox
Pardus 2011 comes with Mozilla Firefox 4.0 Beta9 as the default web browser application. New features of this Firefox release include Firefox Panorama, application tabs, a redesigned extension manager, Jetpack extensions support, integration with Firefox Sync, and support for multitouch displays.
3.3.4.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is set as the default audio playing application as it is well maintained and lightweight.
3.3.4.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.4.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.5. Webbläsare

Although not included in the installation images, a wide range of web browsers including Chromium browser, rekonq, Arora and Opera are available for installation through the package manager.

3.3.6. Systeminställningar

Panda (Pardus alternative driver administration) is a new System Settings module which allows users to change the driver between Open Source and proprietary one for NVIDIA and ATI display adapters.
Users are now able to set up their Bluetooth devices, Wacom tablets and Synaptics touchpads through System Settings.
The default language used throughout the whole system can now be changed through the locale settings section.

3.3.7. Kontorspaket

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is now the default Office Suite in Pardus 2011.

3.3.8. Spel

Pardus 2011 continues the tradition of giving importance to the quality of games offered to their users. The default installation now includes the board and logic games provided by the kdegames package.

3.4. Andra förändringar

Firefox gets new stable release rapidly. 4.0 Beta9 release presents many new features like the new interface of the browser as well as plugin management new interface, tab grouping, bookmarks between computers, passwords and history matching, a new plug-in interface. HTML5, CSS3 and WebM technology supports are also most important features for the developers.

4. Systemkrav

4.1. Processor (CPU)

Du kanske vill installera 64-bitarsversionen av Pardus 2011 om ditt system har någon av processorerna i listan nedan.

4.1.1. Intelprocessorer med stöd för 64-bitarsarkitektur

  • Nyare versioner av Pentium 4, Celeron D, Xeon och Pentium Dual-Core
  • Atom D5xx, N4xx and N5xx
  • Alla versioner av Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 och Core i3 processorer
  • Antagligen processorer tillverkade efter detta dokument skrevs.

4.1.2. AMD-processorer med stöd för 64-bitarsarkitektur

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Antagligen processorer tillverkade efter detta dokument skrevs.

4.2. Systemminne (RAM)

Pardus 2011 krav på systemminnet (RAM) beror starkt av ditt dagliga arbetsflöde så det är svårt att definiera en övre och lägre gräns, men vi rekommenderar minst 1GB minne för en angenäm skrivbordsupplevelse.

4.3. Hårddiskkrav

4.3.1. Minsta krav

Pardus 2011 kräver minst 4GB tillgängligt hårddiskutrymme för en framgångsrik installation. Då vi inte erbjuder individuella val av paket under installationen gäller detta krav alla installationer.

4.3.2. Rekommenderade krav

För att effektivt kunna använda Pardus 2011 på ditt system utan att få ont om diskutrymme under ditt dagliga arbete rekommenderar vi minst 10GB tillgängligt hårddiskutrymme.

4.3.3. Noteras om växlingsutrymme

Växlingsutrymmet används när ditt operativsystem får slut på fysiskt minne. Om du tror att ditt fysiska minne är tillräckligt stort för din dagliga användning kan du hoppa över att skapa växlingsutrymme under installationen.
Lägg märke till att en diskpartition ämnad för växlingsutrymme krävs för att viloläge (Spara till disk) skall fungera. I annat fall är funktionen avstängd.
