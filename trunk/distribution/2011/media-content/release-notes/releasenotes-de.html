Release Notes

1. Willkommen bei Pardus 2011

Willkommen bei Pardus 2011, der neuesten Version einer leistungsfähigen und auf KDE fokussierten Linux Distribution.

1.1. Was ist Pardus?

Pardus ist eine nutzerfreundliche OpenSource GNU/Linux Distribution, die frei verfügbar ist und unter der GPL (General Public License) steht mit vielen nutzerfreundlichen Funktionen, ausgezeichneter Hardwarekompatibilität, herausragender Stabilität und Sicherheit.
Das Pardus Projekt ist finanziert durch das Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), welches der Scientific and Technological Research Council of Turkey (TUBITAK) angegliedert ist.
Das Ziel von Pardus ist, den Benutzern eine stabile, verlässliche, einfach zu nutzende und funktionsreiche Computererfahrung durch moderne und beliebte OpenSource Software zu bieten. Die Entwickler legen großen Wert darauf, dass Pardus auf einer breiten Basis von Hardware läuft.
Das Projekt ist nach dem wissenschaftlichen lateinischen Namen des Anatolischen Leopards, Panthera pardus tulliana benannt.

1.2. Wie kann ich Pardus probieren?

Sie können Pardus testen indem Sie es auf Ihren Computer installieren, oder Sie lassen die Live Version laufen, ohne dass die Festplatte verändert wird. Sie können Pardus von einem optischen Medium genauso wie von einem USB Speicher (ext. Festplatte oder USB-Stick) booten.
Falls Sie ein optisches Medium verwenden wollen, um Pardus zu booten, müssen Sie eine DVD nutzen um das Image zu brennen, da sowohl die Installations- als auch das Live-Image die Größe einer regulären CD übersteigen.
Um von dem Installations- oder Livemedium zu booten sollte die Bootreihenfolge im BIOS Ihres Computers so eingestellt werden, dass zuerst von dem optischen Laufwerk oder vom USB Gerät gebootet wird. Bei neueren Systemen kann die Bootreihenfolge auch durch Drücken einer Taste (bei Desktop Systemen normalerweise F8) während des Bootvorgangs geändert werden.

1.3. Weitere Informationen

Für weitere Informationen können Sie die offizielle Webseite http://www.pardus.org.tr/eng besuchen. 2. Übersicht

Pardus 2011 enthält gegenüber der vorherigen Pardus Version 2009.2 zahlreiche Verbesserungen. Die folgenden Abschnitte erlauben einen kurzen Überblick über die Änderungen.

2.1. Hardwareübersicht

Pardus 2011 unterstützt jetzt die 64-Bit Architektur, welche die neuen Prozessoren von Intel und AMD bieten.

2.2. Softwareübersicht

  • Kernel - Der neueste Linux Kernel 2.6.37 bietet zusammen mit vielen tausend Fehlerkorrekturen eine hochaktuelle Hardwareunterstützung.
  • Plymouth - Die unter Pardus 2009.2 genutzte Bootscreen-Technologie wurde vollständig durch das neue Plymouth ersetzt.
  • YALI - YALI, der Installer von Pardus bietet nun LVM/RAID sowie UUID-Unterstützung.
  • KDE SC - Pardus 2011 wird mit der neuesten KDE Software Compilation, KDE SC 4.5.5 ausgeliefert. Um die Stabilität des Desktops signifikant zu steigern beinhalten die Basispakete zusätzlich eine gewisse Anzahl von Portierungen und Fehlerkorrekturen.
  • Kaptan - Kaptan, das Werkzeug zur Anpassung des Desktops, kann sofern gewünscht ein Foto von Ihnen aufnehmen, und dieses als Avatar in KDE verwenden.
  • NetworkManager - Der GNOME NetworkManager 0.8.2 ist nun der Standardunterbau für das Netzwerk bei Pardus 2011. Die Anwender können nun Ihr HSPA/CDMA/VPN-Netzwerk zusammen mit dem Ethernet- und WLAN-Netzwerk einrichten und nutzen.
  • GTK Oxygen Stil - Alle GTK Programme werden jetzt dank dem oxygen-gtk Project mit dem Oxygen Stil dargestellt. Für die Konsistenz der Benutzeroberfläche ist dies eine sehr große Verbesserung.
  • LibreOffice - LibreOffice, eine OpenSource Suite zur produktiven Office Arbeit, welche durch die Document Foundation gesponsert wird, ist nun das Standard Officeprogramm bei Pardus 2011.

3. Was ist neu?

3.1. Updates der Pardus Technologien

3.1.1. YALI

  • YALI, der Installer von Pardus bietet nun LVM/RAID sowie UUID-Unterstützung.
  • Die Benutzeroberfläche wurde vollständig neu entworfen, um die Nutzerfreundlichkeit zu verbessern.
  • Anwender können nun zwischen OpenSource und proprietären Treibern für ihre NVIDIA Grafikkarte auswählen.

3.1.2. Erster Start

  • Einige der Einstellungen, die während der Installation von Pardus 2009.2 getätigt wurden, sind in eine Anwendung integriert worden, die beim ersten Systemstart nach der Installation ausgeführt wird.

3.1.3. Kaptan

  • Kaptan, das Werkzeug zur Anpassung des Desktops kann sofern gewünscht ein Foto von Ihnen aufnehmen, und dieses als Avatar in KDE verwenden.
  • Anwendern ist es nun möglich ihr bevorzuges Iconset für die KDE Desktopumgebung auszuwählen.
  • Die Benutzeroberfläche wurde vollständig neu entworfen, um die Nutzerfreundlichkeit zu verbessern.

3.1.4. PiSi

  • PiSi nutzt nun intern XZ als Standard Kompressionsverfahren.
  • Der Name der Programmpakete reflektiert nun die Architektur und die Version der Distribution für die sie erstellt wurde.
  • Für eine bessere und schnellere Paketverwaltung wurden verschiedene Verbesserungen in den Kernkomponenten eingeführt.

3.1.5. Paket-Manager

  • Anwendern ist es nun möglich im Paket-Manager ein Programmpaket zu bewerten und einen Screenshot des Programms zu sehen.
  • Um die Bedienung zu vereinfachen wurde die Benutzeroberfläche wurde neu gestaltet. Zusätzlich wurden bei der Paketverwaltung verschiedene Geschwindigkeitsverbesserungen eingeführt.
  • Viel Arbeit wurde in die KDE Integration und der Verbesserung des KDE Systemtrays gesteckt.

3.2. Aktualisierungen der Kernkomponenten

  • Kernel - Der Linux Kernel mit der Version 2.6.37, der mit Pardus 2011 ausgeliefert wird, ist der neueste stabile Kernel der am 05.01.2011 veröffentlicht wurde.
  • Plymouth - Die unter Pardus 2009.2 genutzte Bootscreen-Technologie wurde vollständig durch das neue Plymouth ersetzt.
  • Python - Python ist aktualisiert worden auf 2.7.1.
  • Perl - Perl ist aktualisert worden auf 5.12.2.
  • X.org - Xorg-Server wurde aktualisiert auf 1.9.4 RC1. Durch unsere Entwickler wurden Verbesserungen bei der automatischen Treiberkonfiguration eingeführt.
  • TeX Live - TeX Live das System für Tex-Dokumente ist aktualisiert worden auf 2009.

3.3. Desktoperlebnis

3.3.1. Änderungen der KDE Desktopumgebung

Pardus 2011 wird mit der neuesten KDE Software Compilation, KDE SC 4.5.5 ausgeliefert. Die Basispakete beinhalten zusätzlich eine gewisse Anzahl von Portierungen und Fehlerkorrekturen, um die Stabilität des Desktops signifikant zu steigern.
KDE SC 4.5.5 bietet eine verbesserte Leistung und Stabilität. Die Unterstützung für WebKit, eine verbesserte Darstellung für Netbooks und ein high-speed Cache der schnellen Zugriff auf gespeicherte Ressourcen auf dem Datenträger ermöglicht, sind weitere Vorteile der neuen Version.

3.3.2. Visuelle Verbesserungen

Alle GTK Programme werden nun Dank dem oxygen-gtk Project im Oxygen Stil dargestellt. Für die Konsistenz der Benutzeroberfläche ist dies eine großartige Verbesserung.
Die Iconsammlung Milky wurde speziell für Pardus entworfen und ist im Installationsmedium enthalten. Mit Hilfe von Kaptan oder den Systemeinstellungen kann der Benutzer diese Sammlung verwenden.

3.3.3. Netzwerkmanagement

Nicht nur, dass nun der GNOME NetworkManager der Standard Netzwerkunterbau ist, wird bei Pardus 2011 ein stark modifiziertes und verbessertes KDE Werkzeug zur Netzwerkverwaltung eingesetzt, um dem Anwender eine konsistente Netzwerk Erfahrung zu bieten.
Mit dem neuen Netzwerkunterbau ist es den Anwendern nun möglich ihr Ethernet-, Wireless- (WEP/WPA/WPA2/WPA-EAP) sowie EDGE/3G/CDMA- und VPN-Netzwerk einzurichten.

3.3.4. Standard Programme

3.3.4.1. Mozilla Firefox
Pardus 2011 nutzt als Standard Webbrowser Mozilla Firefox 4.0 Beta9. Diese Version bietet neue Funktionen, wie Firefox Panorama, Applikations Tabs, ein neu gestalteter Addon-Manager, Unterstützung für Jetpack Erweiterungen, die Integration von Firefox Sync und Unterstützung für Multitouch Bildschirme.
3.3.4.2. Clementine
Clementine, ein plattformübergreifender und leichtgewichtiger Musikplayer basierend auf Amarok 1.4 ist das Standard Abspielprogramm für Audioinhalte.
3.3.4.3. SMPlayer
SMplayer, eine auf Qt basierende Oberfläche für MPlayer ist das Standardprogramm für das Abspielen von Video/VCD und DVD. Das Programm ist sehr gut in die KDE Desktopumgebung integriert und bietet neben einer nutzerfreudlichen Bedienung dank dem MPlayer Unterbau eine sehr gute Unterstützung für viele Videoformate.
3.3.4.4. KsCD
KsCD ist das Standardprogramm zum Abspielen von Audio CDs.

3.3.5. Web Browser

Obwohl nicht im Installationsmedium integriert, ist eine große Anzahl von Webbrowsern verfügbar. Unter anderem kann mit dem Paket-Manager Chromium, rekonq, Arora und Opera installiert werden.

3.3.6. Systemeinstellungen

Panda (Pardus alternative driver administration) ist ein neues Systemmodul, das dem Anwender erlaubt zwischen OpenSource und proprietären Treibern für NVIDIA und ATI Grafikarten zu wechseln.
Anwender können nun mit den Systemeinstellungen Bluetoothgeräte, Wacom tablets and Synaptics touchpads konfigurieren.
Die Standard Systemsprache für das gesamte System kann nun leicht durch die Länder/Sprachauswahl geändert werden.

3.3.7. Office Suite

LibreOffice, eine OpenSource Suite zur produktiven Office Arbeit, welche durch die Document Foundation gesponsert wird, ist nun das Standard Officeprogramm bei Pardus 2011.

3.3.8. Spiele

Pardus 2011 führt die Tradition weiter den Anwendern nur Spiele mit hoher Qualität anzubieten. Die Standardinstallation beinhaltet nun die Brett- und Logikspiele, die in dem Paket kdegames integriert sind.

3.4. Weitere Änderungen

Die Firefox-Version 4.0 Beta9 beinhaltet eine Vielzahl an neuen Funktionen wie das neue Benutzerinterface, eine neue Pluginverwaltung, Tabgruppierung, Lesezeichensynchronisierung, Paßwort und Chronik Abgleich. Die Unterstützung von HTML5, CSS3 und der WebM Technologie sind die wichtigsten Funktionen für Entwickler.

4. Systemanforderung

4.1. Prozessor (CPU)

Falls Ihr System mit einem der Prozessoren ausgestattet ist, die in der Sektion unterhalb aufgelistet sind, möchten Sie vielleicht die 64-Bit Version von Pardus 2011 installieren.

4.1.1. Intel Prozessor mit Unterstützung der 64-Bit Architektur

  • Neuere Versionen von Pentium 4, Celeron D, Xeon und Pentium Dual-Core
  • Atom D5xx, N4xx and N5xx
  • Alle Versionen von Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 und Core i3 Prozessoren
  • Möglicherweise ist ein neuerer Prozessor verbaut, der als das Dokument erstellt wurde noch nicht erfasst war.

4.1.2. AMD Prozessor mit Unterstützung der 64-Bit Architektur

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Möglicherweise ist ein neuerer Prozessor verbaut, der als das Dokument erstellt wurde noch nicht erfasst war.

4.2. Arbeitsspeicher (RAM)

Die Anforderung von Pardus 2011 nach Arbeitsspeicher (RAM) hängt stark davon ab, welche arbeiten Sie mit dem System durchführen wollen. Deshalb ist es sehr schwer eine Unter- und Obergrenze zu definieren. Für ein ruckelfreies System empfehlen wir aber wenigstens 1GB Arbeitsspeicher.

4.3. Benötigter Festplattenspeicher

4.3.1. Minimale Anforderung

Für eine erfolgreiche Installation benötigt Pardus 2011 zumindest 4GB freien Speicherplatz. Da wir während der Installation keine Paketauswahl anbieten, ist diese Mindestgröße für alle Installationen gültig.

4.3.2. Empfohlene Anforderung

Um mit Pardus 2011 effektiv arbeiten zu können, ohne ständig durch mangelnden Speicherplatz in Bedrängnis zu kommen, empfehlen wir mindestens 10GB freien Speicherplatz.

4.3.3. Anmerkung zum Swap Speicher

Der Swap Speicher wird benutzt, wenn das System mehr Arbeitsspeicher nutzen möchte, als vorhanden ist. Falls Sie der Meinung sind, dass der verfügbare Arbeitsspeicher groß genug für die tägliche Arbeit ist, können Sie die Erstellung der Swap Partition während der Installation überspringen.
Beachten Sie, dass eine Swap-Partition notwendig ist, falls Sie bei Pardus den Ruhezustand (Suspend to disk bzw. Hibernate) nutzen wollen. Andernfalls ist die Funktionalität nicht möglich.
