Примечания к выпуску

1. Добро пожаловать в Pardus 2011

Добро пожаловать в Pardus 2011, последнюю версию мощного дистрибутива Linux, сфокусированного на использование рабочей среды KDE.

1.1. Что такое Pardus?

Pardus - это полностью открытый, свободно распространяемый, GPL (General Public License) лицензированный, простой в использовании GNU/Linux дистрибутив с множеством усовершенствований в плане удобства использования, совместимости с аппаратным обеспечением, стабильности и безопасности.
Проект Pardus спонсируется Центром исследования новых технологий информатики и информационной безопасности (BILGEM), филиалом Научно-технологического исследовательского совета Турции (TUBITAK).
Цель дистрибутива Pardus в том, чтобы предоставить людям стабильную, надежную, легкую в использовании и многофункциональную систему для работы с компьютером, представленную современным и популярным открытым программным обеспечением. В дополнение ко всему, разработчики прикладывают большие усилия, чтобы Pardus поддерживал широкий спектр оборудования.
Проект назван в честь латинского (и научного) имени Анатолийского леопарда, Panthera pardus tulliana.

1.2. Как я могу попробовать Pardus?

Вы можете попробовать Pardus либо путем установки его на свой компьютер, любо запустив Live-версию, не внося изменения на свой жесткий диск. Вы можете загрузить Pardus с оптического привода, а также с USB-устройств хранения данных, таких как внешние жесткие диски и USB флэш-диски.
В любом случае, установочный и live-образ дистрибутива превышают размер обычного CD-диска, поэтому если вы хотите использовать оптический привод для загрузки Pardus, вы должны использовать для записи образа более емкий DVD-диск.
Обратите внимание, что порядок загрузки в вашей системе (BIOS) должен быть установлен с оптическим приводом или USB-устройством, как первым устройством для загрузки. В новых системах, порядок загрузки можно также изменить, нажав специальную клавишу (как правило, это F8 на настольных системах) во время загрузки.

2. Обзор

Pardus 2011 содержит важные улучшения со времен предыдущего выпуска Pardus 2009.2. В следующих разделах приведен краткий обзор этих изменений.

2.1. Обзор оборудования

Pardus 2011 уже поддерживает 64-битную архитектуру процессора, которая поддерживается многими новейшими процессорами Intel и AMD.

2.2. Обзор программного обеспечения

  • Ядро - Последнее Linux ядро 2.6.37 обеспечивает поддержку современного оборудования, а так же содержит тысячи исправленных ошибок.
  • Plymouth - Технология графической заставки на этапе загрузки, используемая в Pardus 2009.2 полностью переделана и заменена на новый движок Plymouth.
  • YALI - YALI, установщик Pardus, предоставляет поддержку LVM/RAID и UUID.
  • KDE SC - Pardus 2011 поставляется с последней версией KDE Software Compilation, KDE SC 4.5.5. Базовые пакеты так же содержат многочисленные исправления и улучшения, что значительно увеличивает стабильность вашей системы.
  • Kaptan - Kaptan, утилита настройки рабочего стола для Pardus, теперь может (опционально) захватывать изображение с веб-камеры и устанавливать его в качестве аватара в KDE.
  • NetworkManager - GNOME NetworkManager 0.8.2 используется как сетевой модуль по-умолчанию в Pardus 2011. Пользователи теперь могут создавать свои HSPA/CDMA/VPN сети, наряду с уже поддерживаемыми Ethernet и 802.11 WLAN сетями.
  • GTK Oxygen style - Все GTK приложения отображаются со стилем Oxygen, благодаря проектуoxygen-gtk. Это огромный шаг вперед для полноценности и равномерности отображения интерфейса пользователя.
  • LibreOffice - LibreOffice, открытый и свободный офисный пакет, спонсируемый Document Foundation, используется как набор офисных приложений по умолчанию в Pardus 2011.

3. Что нового?

3.1. Обновления в технологиях Pardus

3.1.1. YALI

  • YALI, установщик Pardus, с улучшенной поддержкой LVM/RAID и UUID.
  • Пользовательский интерфейс полностью переработан для улучшения удобства использования.
  • Пользователи теперь могут выбирать между открытыми и проприетарными драйверами для своих видеоадаптеров NVIDIA.

3.1.2. Первая загрузка

  • Некоторые настройки, которые задавались в процессе установки в Pardus 2009.2 теперь перешли в другое приложение, которое появляется один раз в течении первой загрузки установленной системы.

3.1.3. Kaptan

  • Kaptan, утилита настройки рабочего окружения в Pardus, умеет (опционально) захватывать ваше изображение и устанавливать его как аватар в KDE.
  • Пользователи теперь могут выбирать свою любимую тему оформления иконок, которая будет использоваться в их окружении рабочего стола KDE.
  • Пользовательский интерфейс полностью переработан для улучшения удобства использования.

3.1.4. Pisi

  • Pisi теперь использует XZ как алгоритм сжатия по умолчанию.
  • Имена бинарных пакетов теперь отображают архитектуру и релиз дистрибутива, для которого пакет был собран.
  • Были реализованы различные улучшения скорости для основных компонентов системы, ради лучшего управления пакетами.

3.1.5. Пакетный менеджер

  • Пользователи теперь могут голосовать за понравишейся пакет и смотреть скриншоты графических приложений, поставляемые в пакетах.
  • Графический пользовательский интерфейс был полностью переработан для улучшения удобства использования. Были реализованы различные улучшения скорости для лучшего управления пакетами.
  • Было сделано много работы для интеграции KDE, в том числе использования системного лотка KDE.

3.2. Обновления в основных компанентах системы

  • Ядро - Linux ядро, поставляемое с Pardus 2011 - 2.6.37, является новейшим стабильным ядром от 2011/01/05.
  • Plymouth - Технология графической заставки на этапе загрузки, используемая в Pardus 2009.2 полностью переделана и заменена на новый движок Plymouth.
  • Python - Python обновлен до 2.7.1.
  • Perl - Perl обновлен до 5.12.2.
  • X.org - xorg-server обновлен до 1.9.4 RC1, нашими разработчиками было сделано усовершенствование механизма автоматической конфигурации драйвера.
  • TeX Live - документация TeX Live была обновлена до 2009.

3.3. Окружение рабочего стола

3.3.1. Изменения в KDE Desktop Environment

KDE SC - Pardus 2011 поставляется с последней версией KDE Software Compilation, KDE SC 4.5.5. Базовые пакеты так же содержат многочисленные исправления и улучшения, что значительно увеличивает стабильность вашей системы.
KDE SC 4.5.5 обеспечивает более высокую производительность и стабильность для окружения рабочего стола. Среди новых возможностей этой версии можно отметить: поддержка WebKit, улучшенный пользовательский интерфейс для использования на нетбуках, новый высокоскоростной кэш обеспечивает быстрый доступ к ресурсам диска.

3.3.2. Визуальные улучшения

Все GTK-приложения отображаются со стилем Oxygen, благодаря oxygen-gtk проекту. Это огромный шаг вперед для полноценности и равномерности отображения интерфейса пользователя.
Тема иконок Milky, специально созданная для Pardus, поставляется на установочном образе. Пользователи могут установить эту тему иконок через Kaptan или Настройки Системы.

3.3.3. Менеджер сети

Кроме перехода на GNOME NetworkManager как на модуль управления сетью по-умолчанию, Pardus 2011 поставляется с улучшенным и измененным инструментом KDE Network Management, а так же с соответствующим плазмоидом (апплет для KDE Plasma), чтобы дать пользователям больше возможностей для работы с сетью.
С новым сетевым модулем, пользователи могут легко установить и настроить их Ethernet (с поддержкой 802.1x аутентификации), беспроводную 802.11 (с поддержкой WEP/WPA/WPA2/WPA-EAP и 802.1x аутентификации), EDGE/3G/CDMA и VPN сеть.

3.3.4. Стандартные приложения

3.3.4.1. Mozilla Firefox
Pardus 2011 поставляется с Mozilla Firefox 4.0 Beta9 как веб-браузером по-умолчанию. Новые возможности этой версии Firefox включают функцию Firefox Panorama, систему управления вкладками, обновленный менеджер расширений, поддержку расширений Jetpack, интеграцию с Firefox Sync и поддержку для мультитач-дисплеев.
3.3.4.2. Clementine
Clementine, кросс-платформенный музыкальный плеер, созданный в лучших традициях Amarok 1.4, является аудио-плеером по-умолчанию, как хорошо поддерживаемый и легкий.
3.3.4.3. SMPlayer
SMplayer, Qt-фронтенд для MPlayer, установлен по-умолчанию как Video/VCD/DVD плеер, отлично интегрируется в окружение рабочего стола KDE, имеет удобный графический интерфейс и поддерживает воспроизведение широкого спектра типов файлов, благодаря MPlayer.
3.3.4.4. KsCD
KsCD, проигрыватель Audio CD, используемый по-умолчанию.

3.3.5. Веб-браузеры

Помимо содержимого установочного образа, имеется большой спектр поддерживаемых браузеров, включая Chromium, rekonq, Arora и Opera. Они доступны для установки через менеджер пакетов.

3.3.6. Настройки системы

Panda (Pardus alternative driver administration) это новый модуль Настройки системы, который позволяет пользователям менять драйвер на открытый или проприетарный, для NVIDIA и ATI адаптеров дисплея.
Пользователи теперь могут подключать свои Bluetooth устройства, Wacom планшеты и сенсорные панели Synaptics через Настройки Системы.
Язык по-умолчанию, используемый во всей системе, теперь может быть изменен через раздел региональных настроек.

3.3.7. Набор офисных приложений

LibreOffice - LibreOffice, открытый и свободный офисный пакет, спонсируемый Document Foundation, используется как набор офисных приложений по-умолчанию в Pardus 2011.

3.3.8. Игры

Pardus 2011 продолжает традицию радовать своих пользователей ещё и качественными играми. Стандартная установка теперь содержит набор настольных и логических игр пакета kdegames.

3.4. Другие изменения

Совсем скоро выйдет новая стабильная версия Firefox. 4.0 Beta9 представляет множество новых возможностей, таких как новый интерфейс браузера, а также плагин управления новым интерфейсом, группировка вкладок, синхронизация паролей, истории и закладок между компьютерами. Поддержка HTML5, CSS 3 и WebM и наиболее важных функций для разработчиков.

4.2. Оперативная память (RAM)

Требуемый объем оперативной памяти (RAM) для Pardus 2011 сильно зависит от вашей ежедневной нагруженности системы, поэтому очень трудно определить нижнюю и верхнюю границу значений, но мы рекомендуем по крайней мере 1GB оперативной памяти для хорошей работы системы и рабочего стола.Требуемый объем оперативной памяти (RAM) для Pardus 2011 сильно зависит от вашей ежедневной нагруженности системы, поэтому очень трудно определить нижнюю и верхнюю границу значений, но мы рекомендуем по крайней мере 1GB оперативной памяти для хорошей работы системы и рабочего стола.

4.3. Требования к дисковому пространству

4.3.1. Минимальные системные требования

Pardus 2011 требует, по крайней мере 4GB свободного дискового пространства для успешной установки. Так как мы не предлагаем выбор устанавливаемых пакетов в процессе инсталляции, то это ограничение актуально для всех установок.

4.3.2. Рекомендуемые системные требования

С целью эффективного использования Pardus 2011 на вашей системе без нехватки свободного дискового пространства, во время вашей ежедневной работы за компьютером, мы рекомендуем как минимум 10GB свободного места на жестком диске.

4.3.3. О разделе подкачки (swap)

Раздел подкачки используется всякий раз, когда вашей операционной системе не хватает физической памяти. Если вы считаете, что размер физической памяти на вашем компьютере достаточно велик для ежедневной работы, то можете пропустить шаг создания раздела подкачки во время установки.
Обратите внимание, что раздел диска, отведенный под раздел подкачки необходим,если вы хотите использовать спящий режим в Pardus. В противном случае, данная функция будет отключена.
