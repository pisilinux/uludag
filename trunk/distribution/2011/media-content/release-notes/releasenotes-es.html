Notas de Lanzamiento

1. Bienvenido a Pardus 2011

Bienvenido a Pardus 2011, la versión más reciente de Pardus, una distribución de Linux potente, con enfoque al entorno de escritorio KDE.

1.1. Qué es Pardus ?

Pardus es un proyecto de una distribución de GNU/Linux amigable al usuario, creado con código abierto, a libre distribución, y liecnciado pajo la licencia GPL (General Public License) , con varios mejoras en usabilidad, compatibilidad de hardware, estabilidad y seguridad.
El auspiciante (sponsor) del proyecto Pardus es el Centro de Investigación de Tecnologías Avanzadas de Informática y Seguridad Informática (BILGEM), un asociado del Concilio de Investigación Científica y Tecnológica de Turquía (TUBITAK).
La distribución Pardus se esfuerza para brindar a la gente una experiencia de computación estable, confiable y fácil de usar mediante la provisión de software de código abierto y popular. Además los desarrolladores trabajan arduamente para lograr una compatibilidad con una amplia gama de hardware.
El nombre del proyecto viene del nombre en latín (cientifico) del leopardo de Anatolia, Panthera pardus tulliana.

1.2. Como puedo probar Pardus ?

Para usar Pardus puede instalarlo en su computadora o ejecutando el sistema en la versión live (en vivo) sin modificar su disco duro. Puede bootear Pardus con un lector óptico o de un dispositivo de almacenamiento USB como discos externos o Pendrive.
Tanto la versión liva como la versión de instalación superan el tamaño de una CD común, por lo tanto si prefiere utilizar un medio óptico para bootear Pardus, debe grabar las imagenes en DVD.
Tenga en cuenta que debe asegurarse que en el orden de arranque en el BIOS el lector óptico o el dispositivo USB deben estar en primer posición para arrancar de ello tanto para la instalación como en modo vivo. En algunos sistemas se puede seleccionar el orden de dispositivos de arranque apretando una tecla (generalmente F8 para equipos de escritorio) durante el arranque.

2. Vista General

Pardus 2011 contiene muchas mejoras en relación a la versión anterior de Pardus 2009.2. Las siguientes secciones muestran una breve información sobre las modificaciones.

2.1. Hardware

Pardus 2011 soporta ahora arquitecturas con procesadores de 64-bit que soportan muchos procesadores Intel y AMD.

2.2. Software

  • Kernel - El último kernel 2.6.37 provee un soporte de hardware más amplio y muchas correcciones de errrores.
  • Plymouth - Se dejó de utilizar la tecnología bootsplash de Pardus 2009.2, en su reemplazo se emplea ahora la nueva Plymouth engine.
  • YALI - El asistente de instalación, YALI, ahora viene con soporte para LVM/RAID y UUID.
  • KDE SC - Pardus 2011 le trae la última Compilación de KDE Software, KDE SC 4.5.5. EL paquete base además incluye varios códigos backports y correcciones para mejorar notablemente la estabilidad de su escritorio.
  • Kaptan - El asistente de primer configuración, Kaptan, tien ahora una opción de sacarle una foto para usarlo como su avatar de acceso en KDE.
  • NetworkManager - GNOME NetworkManager 0.8.2 es ahora empleado como herramienta de conectividad en Pardus 2011. Usuarios pueden ahora configurar sus redes HSPA/CDMA/VPN en conjunto con las redes ya soportados de Ethernet y 802.11 WLAN.
  • GTK Oxygen style - Todas la aplicaciones GTK ahora se renderean al estilo Oxygen gracias al proyecto oxygen-gtk. Esto mejora mucho la consistencia de la interfaz de usuario.
  • LibreOffice - LibreOffice, la suite de oficina de código abierta auspiciada por la Document Foundation es ahora la suite ofimatica predeterminada de Pardus 2011.

3. Que novedades trae ?

3.1. Actualizaciones de tecnologías de Pardus

3.1.1. YALI

  • YALI, el instalador de Pardus, viene con soporte LVM/RAID y UUID.
  • La interfaz del usuario fue totalmente rediseñado para mejorar la usabilidad
  • Usuarios pueden ahora elegir entre controladores de código abierto o propietarios para sus adaptadores de video NVIDIA.

3.1.2. Primer Arranque

  • Parte de los parametros que en Pardus 2009.2 se ingresaron durante la instalación, ahora se ingresan en un programa diferente, que aparece al primer arranque.

3.1.3. Kaptan

  • Kaptan, el asistente de configuración del escritorio de Pardus, ahora puede tomarle una foto y usarlo como avatar en KDE.
  • Usuarios pueden elegir su tema de íconos para el entorno de escritorio KDE.
  • La interfaz del usuario fue totalmente rediseñado para mejorar la usabilidad

3.1.4. Pisi

  • Pisi ahora usa internamente el algoritmo XZ predeterminado para su método de compresión.
  • Los nombres de los paquetes binarios ahora reflejan la arquitectura y la versión de la distribución correspondiente.
  • Varias mejoras se aplicaron en componentes internos, para una mejor experiencia con el administrador de paquetes.

3.1.5. Administrador de Paquetes

  • Usuarios ahora pueden dar su voto de calificación al paquete, y ver una captura de pantalla de los aplicativos gráficos.
  • La interfaz de usuario fue rediseñada para una mejor usabilidad. Se realizaron varias mejoras para aumentar la velocidad y una mejor experiencia visual.
  • Se realizaron muchos trabajos de integración con el escritorio KDE, incluyendo el uso de la bandeja del sistema.

3.2. Actualizaciones en componentes internos

  • Kernel - La versión del kernel incluído en Pardus 2011 es 2.6.37 que es la última versión estable del kernel, publicado el 2011/01/05.
  • Plymouth - Se dejó de usar la tecnología de bootsplash usada en Pardus 2009.2, se reemplazó por el nuevo motor Plymouth.
  • Python - Python actualizado a 2.7.1.
  • Perl - Perl actualizado a 5.12.2.
  • X.org - xorg-server actualizado a 1.9.4 RC1, además nuestros desarrolladores realizaron mejoras en el mecanismo de configuración automático.
  • TeX Live - TeX Live documentation stack actualizado a 2009.

3.3. Experiencia de Escritorio

3.3.1. Modificaciones del entorno de escritorio KDE

Pardus 2011 comes with the latest KDE Software Compilation, KDE SC 4.5.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE SC 4.5.5 brings more performance and stability to the desktop experience. Support for WebKit, improved user experience for netbooks, a new high-speed cache allowing fast access to resources stored on disk are among the new features of this version.

3.3.2. Mejoras Visuales

All GTK applications are now rendered with Oxygen style thanks to the oxygen-gtk project. This brings a huge improvement to the user interface consistency.
Milky icon theme especially designed for Pardus is shipped within the installation image. Users can set this icon theme through Kaptan or System Settings.

3.3.3. Configuración de Red

Besides switching to GNOME NetworkManager as the default networking backend, Pardus 2011 comes with a heavily improved and modified KDE Network Management tool and plasmoid to give their users a much more consistent networking experience.
With this new networking backend, users are now able to easily set up their Ethernet (With support for 802.1x authentication), 802.11 Wireless (With support for WEP/WPA/WPA2/WPA-EAP and 802.1x authentication), EDGE/3G/CDMA and VPN networks.

3.3.4. Aplicaciones predeterminadas

3.3.4.1. Mozilla Firefox
Pardus 2011 comes with Mozilla Firefox 4.0 Beta9 as the default web browser application. New features of this Firefox release include Firefox Panorama, application tabs, a redesigned extension manager, Jetpack extensions support, integration with Firefox Sync, and support for multitouch displays.
3.3.4.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is set as the default audio playing application as it is well maintained and lightweight.
3.3.4.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.4.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.5. Navegadores Web

Although not included in the installation images, a wide range of web browsers including Chromium browser, rekonq, Arora and Opera are available for installation through the package manager.

3.3.6. Configuración del sistema

Panda (Pardus alternative driver administration) is a new System Settings module which allows users to change the driver between Open Source and proprietary one for NVIDIA and ATI display adapters.
Users are now able to set up their Bluetooth devices, Wacom tablets and Synaptics touchpads through System Settings.
The default language used throughout the whole system can now be changed through the locale settings section.

3.3.7. Suite Offimatica

LibreOffice, una suite de ofimatica de código abierto, auspiciado por la Document Foundation ahora es la suite ofimatica predeterminada de Pardus 2011.

3.3.8. Juegos

Pardus 2011 continues the tradition of giving importance to the quality of games offered to their users. The default installation now includes the board and logic games provided by the kdegames package.

3.4. Other Changes

Firefox gets new stable release rapidly. 4.0 Beta9 release presents many new features like the new interface of the browser as well as plugin management new interface, tab grouping, bookmarks between computers, passwords and history matching, a new plug-in interface. HTML5, CSS3 and WebM technology supports are also most important features for the developers.

4. Requirimientos al Sistema

4.1. Procesador (CPU)

Le convendría instalar la versión de 64-bit de Pardus 2011 si sus sistema está equipado con uno de los procesadores del listado abajo.

4.1.1. Procesadores INTEL con soporte a la arquitectura 64-bit

  • Versiones más reciente de Pentium 4, Celeron D, Xeon y Pentium Dual-Core
  • Atom D5xx, N4xx and N5xx
  • Todas las versiones de Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 y Core i3
  • Probablemente procesadores con fabricación más reciente que este informativo.

4.1.2. Procesadores AMD con soporte a la arquitectura 64-bit

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Probablemente procesadores con fabricación más reciente que este informativo.

4.2. Memoria del sistema (RAM)

Los requirimientos de memoria RAM de Pardus 2011 dependen del tipo de uso que le dan al sistema, y por lo tanto es dificil de determinar límites inferiores o superiores. Sin embargo recomendamos al mínimo 1GB de RAM para una experiencia agradable del escritorio.

4.3. DIsco rígido

4.3.1. Requisitos mínimos

Pardus 2011 requiere al menos 4GB de espacio en disco para uns instalación exitosa. Este valor de espacio está predeterminado por la cantidad de paquetes que se van a instalar, ya que durante de la instalación no se tiene la opción de elegir los paquetes a instalar.

4.3.2. Requirimientos recomendados

Para un uso eficiente de Pardus 2011 en su sistema sin llegar al límite de espacio en disco para su uso diario, recomendamos al menos 10GB de espacio disponible en disco.

4.3.3. Comentario sobre espacio Swap

El sistema usa el espacio de intercambio (Swap), cuando la memoria RAM del sistema ya no da abasto. Si cree que tiene suficiente memoria RAM para su uso diario, puede omitir la creación del espacio de intercambio (swap) durante la instalación.
Atención: Si desea utilizar la funcionalidad de hibernar (Suspend to disk) en Pardus se necesita la existencia del espacio de intercambio Pardus. Esta función queda deshabilitada si no existe el espacio de intercambio.
