Versionsfakta

1. Welcome to Pardus 2011.1 Dama dama

Welcome to Pardus 2011.1 Dama dama, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Vad är Pardus?

Pardus är öppenkällkod, fritt distribuerbar, GPL (General Public License)-licensierad användarvänlig GNU/Linuxdistribution med många förbättringar inom användbarhet, hårdvarukompatibilitet, stabilitet och säkerhet.
Pardusprojektet sponsras av Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), ett dotterbolag till Scientific and Technological Research Council of Turkey (TUBITAK).
Målet med Pardusdistributionen är att erbjuda människor en stabil, pålitlig, lättanvänd och innehållsrik datorupplevelsegenom att erbjuda moderna och populära program som har öppen källkod. Utvecklarna lägger också extra kraft på att få Pardus körbar på ett brett spektrum av hårdvara.
Projektet är döpt efter det latinska och vetenskapliga namnet på den anatoliska leoparden, Panthera pardus tulliana.

1.2. Hur kan jag testa Pardus?

Du kan testa Pardus antingen genom att installera det på din dator eller köra det live utan att påverka din hårddisk. Du kan starta Pardus från en optisk enhet eller från USB-enheter såsom hårddiskar och minnepinnar.
Både installationen och liveversionens avbilder överstiger vad som får plats på en vanlig CD, om du vill använda en optisk enhet för att starta Pardus måste du för närvarande använda en DVD.
Tänk på att uppstartsordningen i datorns BIOS-program måste ställas in till att först starta från optisk enhet eller USB-enhet för att start av live eller installation skall ske. På nyare system kan uppstartsordningen också ställas in genom att trycka på en tangent, vanligtvis F8, under uppstart.

2. Översikt

Pardus 2011.1 innehåller stora förbättringar jämfört med föregående Pardusutgåva 2011. Här följer en kort sammanfattning över dessa förbättringar.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundred of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.1 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.1.

3. Vad är nytt?

3.1. Uppdateringar av Pardustekniker

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Uppdateringar av kärnkomponenter

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Skrivbordsupplevelse

3.3.1. Changes in KDE Desktop Environment

Pardus 2011.1 comes with the latest KDE Software Compilation, KDE SC 4.6.5. The base packages also contains numerous backports and fixes which will improve the stability of your desktop experience significantly.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Förvalda program

3.3.2.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, a Qt based frontend for MPlayer is set as the default Video/VCD/DVD playing application as it is well integrated with KDE desktop environment in terms of usability and graphical user interface and supports playing a wide range of video file types thanks to the MPlayer backend.
3.3.2.4. KsCD
KsCD, is set as the default Audio CD playing application.

3.3.3. Webbläsare

Although not included in the installation images, a wide range of web browsers including Chromium browser, rekonq, Arora and Opera are available for installation through the package manager.

3.3.4. Systeminställningar

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Kontorspaket

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.1. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.1 package repopsitories.

4. Systemkrav

4.1. Processor (CPU)

Du kanske vill installera 64-bitarsversionen av Pardus 2011.1 om ditt system har någon av processorerna i listan nedan.

4.1.1. Intelprocessorer med stöd för 64-bitarsarkitektur

  • Nyare versioner av Pentium 4, Celeron D, Xeon och Pentium Dual-Core
  • Atom D5xx, N4xx and N5xx
  • Alla versioner av Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 och Core i3 processorer
  • Antagligen processorer tillverkade efter detta dokument skrevs.

4.1.2. AMD-processorer med stöd för 64-bitarsarkitektur

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Antagligen processorer tillverkade efter detta dokument skrevs.

4.2. Systemminne (RAM)

Pardus 2011.1 krav på systemminnet (RAM) beror starkt av ditt dagliga arbetsflöde så det är svårt att definiera en övre och lägre gräns, men vi rekommenderar minst 1GB minne för en angenäm skrivbordsupplevelse.

4.3. Hårddiskkrav

4.3.1. Minsta krav

Pardus 2011.1 kräver minst 4GB tillgängligt hårddiskutrymme för en framgångsrik installation. Då vi inte erbjuder individuella val av paket under installationen gäller detta krav alla installationer.

4.3.2. Rekommenderade krav

För att effektivt kunna använda Pardus 2011.1 på ditt system utan att få ont om diskutrymme under ditt dagliga arbete rekommenderar vi minst 10GB tillgängligt hårddiskutrymme.

4.3.3. Noteras om växlingsutrymme

Växlingsutrymmet används när ditt operativsystem får slut på fysiskt minne. Om du tror att ditt fysiska minne är tillräckligt stort för din dagliga användning kan du hoppa över att skapa växlingsutrymme under installationen.
Lägg märke till att en diskpartition ämnad för växlingsutrymme krävs för att viloläge (Spara till disk) skall fungera. I annat fall är funktionen avstängd.
