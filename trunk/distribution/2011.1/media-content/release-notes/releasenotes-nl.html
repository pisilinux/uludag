1. Welcome to Pardus 2011.1 Dama dama

Welcome to Pardus 2011.1 Dama dama, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Wat is Pardus?

Pardus is een open-source, vrij beschikbare, onder GPL-licentie (General Public License) vallende, gebruikersvriendelijke GNU/Linux-distributie met veel gebruikersgemak, hardware-compatibiliteit, stabiliteit en veilige mogelijkheden.
Het Pardus-project wordt ondersteund door de Center of Research for Advanced Technologies of Informatics and Information Security (BILGEM), een afdeling van de Scientific and Technological Research Council of Turkey (TUBITAK).
Het doel van de Pardus-distributie is het aanbieden van een stabiele, betrouwbare, gemakkelijk-in-gebruik en vele-mogelijkheden computerervaring door gebruik te maken van moderne en populaire open-source software. De ontwikkelaars hebben tevens additionele inspanningen gedaan om Pardus op zoveel mogelijk hardware mogelijk te maken.
Het project is genoemd naar de Latijnse (en wetenschappelijke) naam van de Anatolische luipaard, Panthera pardus tulliana.

1.2. Hoe kan ik Pardus proberen?

Het is mogelijk om Pardus te proberen door het op de computer te installeren of door het starten van de live-versie en daardoor de inhoud van de harde schijf niet te veranderen. Pardus kan opgestart worden vanaf een optisch medium en ook vanaf een USB-medium zoals externe harde schijf en USB-geheugen/stick.
Zowel de installatie- en ook de live-uitgave passen niet op een normale CD en dus is het gebruik van een DVD noodzakelijk om Pardus via een optisch medium te gebruiken.
Let op dat de volgorde van opstarten in de BIOS van de computer ingesteld dient te worden met het optisch - of USB-station als eerste opstartmedium. Bij de meeste nieuwere computers kan deze volgorde ook geselecteerd worden door het indrukken van een toets (meestal F8 of F11) tijdens het opstarten.

2. Overzicht

Pardus 2011.1 bevat belangrijke verbeteringen t.o.v. de vorige uitgave van Pardus 2011. De volgende sectie geven een korte samenvatting van deze wijzigingen.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundred of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.1 komt met de laatste softwarecompilatie van KDE, KDE SC 4.6.5. Het basispakket bevat ook vele toepassingen en foutoplossingen, die de ervaring met en de stabiliteit van het systeem in belangrijke mate zal verbeteren.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.1.

3. Wat is er nieuw ?

3.1. Vernieuwingen in de technologie van Pardus

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Verbeteringen in de hoofdcomponenten

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Bureaubladervaring

3.3.1. Wijzigingen in de KDE-omgeving

Pardus 2011.1 komt met de laatste softwarecompilatie van KDE, KDE SC 4.6.5. Het basispakket bevat ook vele toepassingen en foutoplossingen, die de ervaring met en de stabiliteit van het systeem in belangrijke mate zal verbeteren.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Standaard-toepassingen

3.3.2.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, een op QT-gebaseerde interface voor MPlayer is ingesteld als de standaard Video/VCD/DVD-toepassing voor afspelen en uitstekend geïntegreerd met KDE betreffende gebruik en grafische gebruikersinterface en ondersteuning voor het afspelen van een uitgebreide range van videoformaatbestanden dankzij de MPlayer-backend.
3.3.2.4. KsCD
KsCD, is ingesteld als de standaard geluid-cd afspelertoepassing.

3.3.3. Web-browsers

Alhoewel niet in het installatiemedium opgenomen zijn vele web-browsers inclusief Chromium-browser, rekonq, Arora en Opera beschikbaar in de pakketbeheerder.

3.3.4. Systeeminstellingen

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Kantoortoepassingen

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.1. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.1 package repopsitories.

4.2. Systeemgeheugen (RAM)

De eisen voor systeemgeheugen (RAM) van Pardus 2011.1 zijn sterk afhankelijk van het dagelijks gebruik en daarom is de boven- en ondergrens moeilijk aan te geven, ten minste 1GB geheugen wordt aanbevolen voor een prettige bedieningservaring.

4.3. Eisen voor harde schijfruimte

4.3.1. Minimum eisen

Pardus 2011.1 heeft tenminste 4GB vrije ruimte op de harde schijf nodig voor een succesvolle installatie. Omdat het niet mogelijk is het pakket tijdens de installatie aan te passen, is dit de minimale eis voor alle installaties.

4.3.2. Aanbevolen eisen

Voor het comfortabel dagelijks werken met Pardus 2011.1 en daarbij het risico van onvoldoende schijfruimte te vermijden wordt tenminste 10GB vrije hard schijfruimte aanbevolen.

4.3.3. Opmerking over swapruimte

Swapruimte is nodig wanneer het besturingssysteem over onvoldoende systeemgeheugen (RAM) beschikt. Wanneer het aanwezige systeemgeheugen voldoende is voor het dagelijks gebruik, is het aanmaken van swapruimte niet nodig tijdens de installatie.
Let op: een gereserveerde partitie voor swapruimte is noodzakelijk indien de schijf-slaapstand (Suspend to disk) in Pardus gebruikt wordt. Anders is deze functie uitgeschakeld.
