Notes de version

1. Welcome to Pardus 2011.1 Dama dama

Welcome to Pardus 2011.1 Dama dama, the latest maintenance release of Pardus, a powerful, KDE-focused Linux distribution.

1.1. Qu'est-ce que Pardus ?

Pardus est une distribution Linux librement distribuable et modifiable sous les termes de la licence publique générale GNU GPL, mettant l'accent sur l'ergonomie, la prise en charge du matériel, la stabilité et la sécurité.
Le projet Pardus est sponsorisé par le Centre de Recherche en Technologies Informatiques de pointe et Sécurité de l'Information (BILGEM), affilié au Conseil de recherche Scientifique et Technologique de Turquie (TUBITAK).
L'objectif de Pardus est de fournir à l'utilisateur de base une distribution stable, fiable, facile à utiliser, riche en fonctionnalités, et libre. Un effort particulier est fait pour que Pardus tourne sur une large gamme de matériel.
Pardus est le nom scientifique (latin) du léopard d'Anatolie, Panthera pardus tulliana.

1.2. Comment puis-je utiliser Pardus ?

Vous pouvez soit installer Pardus sur le disque dur de votre ordinateur, soit utiliser la version Live sans toucher à votre disque dur. Pardus peut démarrer sur un DVD ou sur une clef ou disque dur USB.
La version installable comme la version Live excèdent la capacité d'un CDROM, par conséquent, il faut graver l'image sur DVD pour utiliser Pardus à partir d'un lecteur optique.
Notez que l'ordre de démarrage du BIOS de votre ordinateur doit donner la priorité au lecteur de DVD ou au périphérique de stockage USB plutôt qu'au disque dur. Vous pouvez accéder aux paramètres du BIOS en appuyant sur une touche (DEL, F8, ou autre selon le fabricant) lors des premières secondes du démarrage de votre PC.

2. Vue d'ensemble

Pardus 2011.1 contient des changements majeurs par rapport à la version 2011 précédente. Les sections suivantes en donnent un bref aperçu.
  • Kernel - 2.6.37.6 Linux kernel provides updated hardware support together with a hundred of bug fixes.
  • YALI - YALI, the installer of Pardus, gained Rescue Mode which provides easy password and bootloader recovery.
  • KDE SC - Pardus 2011.1 est livrée avec le dernier KDE, KDE SC 4.6.5. Les paquets de base contiennent des rétro-portages et des correctifs qui améliorent significativement la stabilité de votre bureau.
  • LibreOffice - LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is updated to the much-awaited 3.4 version which brings huge improvements over the older version.
  • Firefox - Firefox 5.0, the latest stable version of the most popular Open Source web browser, is now shipped with Pardus 2011.1.

3. Ce qui change

3.1. Les technologies de Pardus

3.1.1. YALI

  • YALI, the installer of Pardus, gained system rescue menu that provides easy bootloader and password recovery.

3.1.2. Pisi

  • Some of the core components are now parallellized to increase the utilization of multi-core machines which means faster package operations and better package management experience.

3.1.3. QuickFormat

  • QuickFormat is a removable device formatting tool that can integrate to Dolphin. Now, it's very easy to format USB flash drives and card readers.

3.2. Mises à jour des composants de base

  • Kernel - The Linux kernel shipped with Pardus 2011.1 is 2.6.37.6.
  • Plymouth - The bootsplash technology used in Pardus 2011 is updated to its latest version for a much stable experience.
  • X.org - xorg-server is updated to 1.9.5.
  • TeX Live - TeX Live documentation stack is updated to latest 2011 version.

3.3. Environnement de bureau

3.3.1. Changements dans l'environnement de bureau KDE

Pardus 2011.1 est livrée avec le dernier KDE, KDE SC 4.6.5. Les paquets de base contiennent des rétro-portages et des correctifs qui améliorent significativement la stabilité de votre bureau.
KDE Software Compilation, versioned 4.6.5, provide many new features in each of KDE's components including Plasma, Solid, Nepomuk, KWin and also introduces a brand new Bluetooth framework called BlueDevil allows easy device pairing and management.

3.3.2. Applications par défaut

3.3.2.1. Mozilla Firefox
Pardus 2011.1 comes with Mozilla Firefox 5.0 as the default web browser application. This version of Firefox introduces CSS animations, Do-Not-Track, improved support for HTML5, MathML, and SMIL, improved network, memory and JavaScript performance and fixes for several stability and security issues.
3.3.2.2. Clementine
Clementine, a cross-platform music player based on Amarok 1.4 is updated to latest stability and maintenance version 0.7.1.
3.3.2.3. SMPlayer
SMplayer, l'habillage graphique Qt de MPlayer a été choisi comme lecteur vidéo, VCD et DVD par défaut compte tenu de son intégration dans KDE en terme d'utilisabilité, d'interface graphique et de sa large gamme de types de formats pris en charge par MPlayer.
3.3.2.4. KsCD
KsCD est l'application par défaut pour lire les CD.

3.3.3. Navigateur web

Sont disponibles en sus, dans le gestionnaire de paquets, les navigateurs Chromium, Rekonq, Arora et Opera.

3.3.4. Paramètres du système

The display configuration tool of the graphics card manufacturer is now recommended in the Display and Monitor module of the System Settings, if it suits better to your graphics card driver.

3.3.5. Suite bureautique

LibreOffice, an Open Source personal productivity suite sponsored by the Document Foundation, is the default Office Suite in Pardus 2011.1. 3.4 version of LibreOffice introduces; improved text rendering, configurable color charts, improved thumbnailed HTML export of presentations, faster Flat ODF import/export and better desktop theme integration.

3.3.6. Other

Most wanted packages like Skype, a P2P VoIP client that allows you to make voice calls, and Wine, a software that allows you to run Microsoft Windows applications, are now included in Pardus 2011.1 package repopsitories.

4. Pré-requis du système

4.1. Processeur (CPU)

Vous pouvez installer la version 64-bit de Pardus 2011.1 si votre système est équipé des processeurs listés dans les sections suivantes.

4.1.1. Processeurs Intel 64-bit

  • Nouvelles versions de Pentium 4, Celeron D, Xeon et Pentium Dual-Core
  • Atom D5xx, N4xx et N5xx
  • Processeurs Pentium D, Pentium Extreme Edition, Core 2, Core i7, Core i5 et Core i3
  • Processeurs plus récents que la date de rédaction de ce document.

4.1.2. Processeurs AMD 64-bit

  • AMD Athlon 64, 64 X2, 64 FX
  • AMD Athlon II
  • AMD Opteron
  • AMD Turion 64
  • AMD Turion 64 X2
  • AMD Sempron
  • AMD Phenom
  • AMD Phenom II
  • Processeurs plus récents que la date de rédaction de ce document.

4.2. Mémoire vive (RAM)

Les pré-requis en mémoire vive de Pardus 2011.1 dépendent de l'intensité de votre utilisation de l'ordinateur. Nous recommandons au moins 1GB de mémoire vive.

4.3. Pré-requis de disque dur

4.3.1. Pré-requis minimum

Pardus 2011.1 nécessite au moins 4GB d'espace disque pour son installation.

4.3.2. Pré-requis recommandés

Afin d'utiliser Pardus 2011.1 sans être à court d'espace disque, nous recommandons au moins 10GB d'espace libre.

4.3.3. Note sur la swap

L'espace d'échange (SWAP) est utilisé par le système à court de mémoire physique. Si vous pensez que la capacité de votre mémoir physique est suffisante pour votre usage quotidien, vous pouvez vous passer de swap.
Notez que une partition réservée à la swap est requise si vous souhaitez utiliser la fonction d'hibernation (Suspend to disk) de Pardus. Autrement, cette fonctionnalité sera désactivée.
